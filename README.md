# TouchType
Lightweight typing practice application for macOS!

![Screenshot](tt2.png)


# Built with:
- Swift <br>
- AppKit <br>
- NSView <br>
- Core Graphics <br>

# Special Feautures:
- Words per Minute calculation <br>
- Custom Practice Input <br>
- Minimalist design <br>
